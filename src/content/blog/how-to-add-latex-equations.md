Embedding mathematical formulas is crucial for academic writing. Here is how you can do it in AstroScholar.

## Inline Equations

You can add inline equations using standard LaTeX syntax.

## Block Equations

For more complex formulas, use block equations.

Einstein's famous equation is defined as:

$$E = mc^2$$

You can also write more complex formulas, such as the quadratic formula:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
